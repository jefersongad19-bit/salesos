# SalesOS v2 · PWA

## ⚡ Deploy no GitHub Pages

### 1. Acesse seu repositório
`github.com/jefersongad19/salesos`

### 2. Apague os arquivos antigos
- Clique em cada arquivo → botão lixeira → commit

### 3. Faça upload dos novos arquivos
Arraste para o repositório (raiz):
- `index.html`
- `sw.js`  
- `manifest.json`
- pasta `icons/` com icon-192.png e icon-512.png

### 4. Confirme que o GitHub Pages está ativo
Settings → Pages → Source: `main` / `/ (root)` → Save

### 5. Acesse:
`https://jefersongad19.github.io/salesos`

---

## 📱 Instalar como app nativo

**Android:** O banner de instalação aparece automaticamente após 3 segundos

**iPhone:** Safari → Compartilhar → "Adicionar à Tela de Início"

**PC:** Ícone de instalar na barra do Chrome/Edge
